## 실행 방법

```bash
npm install
npm run dev
```

`http://localhost:5173` 접속 후 플레이. WASD/방향키로 이동.

## 배포

```bash
npm run deploy
```

`dist/` 빌드 결과물을 `gh-pages` 브랜치로 푸시하여 GitHub Pages에 배포합니다.

## 커밋 컨벤션

커밋 메시지 앞에 작성 주체를 태그로 표기합니다.

| 태그 | 의미 |
| --- | --- |
| `[AI]` | AI가 작성한 코드/문서 |
| `[HAND]` | 사람이 직접 작성 |
| `[MIX]` | AI 초안을 사람이 수정하는 등 혼합 작업 |

예: `[AI] spike: Phaser 뱀서 코어 + 전투 완성`
